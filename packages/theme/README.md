# Qbook
[live demo](https://vuepress-theme-qbook.vercel.app/)
- 支持同时启用博客+文档页面
- 首页轮播大图(支持蒙版)
- 标签分类
- 卡片式文章展示
- markdown增强
  - 换行
  - 脚注
  - 高亮
  - emoji
  - TOC目录
- 自定义容器
- 通知
- 文档侧边栏目录
- 代码块复制按钮
- katex公式支持
- mermaid代码块支持
